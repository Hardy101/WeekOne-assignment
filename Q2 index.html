Table Tags Names of Tags Definition Text Format
< b > Bold Tag Defines bold Text without extra importance content is displayed in bold
< strong > Strong Tag Defines text with strong importance content is displayed in bold
< i > Italic Tag Defines part of text in an alternate voice or mood content is displayed in italic
< em > Emphasis Tag Defines an emphasized text content is displayed in italic
< mark > Mark Tag Defines text that should be highlighted content is displayed in a yellow highlight by default
< small > Small Tag Defines smaller text content is displayed in a smaller text
< del > Deleted Tag Defines text that has been deleted from a document content will is be displayed in strikethrough
< ins > Inserted Tag Defines a text that has been inserted content will be underlined
< sub > Subscript Tag Defines a Subscript text content will be displayed below normal line and as a smaller text
< sup > Superscript Tag Defines a Superscript text content will be displayed above normal line and as a smaller text
